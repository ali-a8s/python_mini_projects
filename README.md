# Mini Python Projects 


- [Rock_Paper_Scissors](https://github.com/ali-a8s/python_mini_projects/blob/main/Rock_Paper_Scissors.py): a simple terminal-based Rock Paper Scissors game with interactive ASCII graphics and score tracking. It features animated countdowns, input validation, and a persistent scoreboard. Play multiple rounds against the computer in a fun, console experience.

- [Password](https://github.com/ali-a8s/python_mini_projects/blob/main/Password.py): a secure and customizable password generator with two modes: default (strong passwords with all character types) and custom (user-selected character groups).

- [BMI_calculator](https://github.com/ali-a8s/python_mini_projects/blob/main/BMI_calculator.py): a simple command-line BMI calculator that computes your body mass index and categorizes the result.

- [Calculator](https://github.com/ali-a8s/python_mini_projects/blob/main/Calculator.py): a versatile terminal calculator supporting basic arithmetic operations, multi-number calculations, averages, and exponents.

- [Number Guesting](https://github.com/ali-a8s/python_mini_projects/blob/main/Number_Guesting.py): an interactive number guessing game where players try to find a random number within a custom range. Features adjustable difficulty levels (easy/hard), smart range validation, and real-time feedback with remaining chances displayed after each guess.
